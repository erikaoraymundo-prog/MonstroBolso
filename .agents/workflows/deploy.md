---
description: Como fazer o deploy para GitHub Pages
---

Para publicar o jogo no GitHub Pages:

1. Faça o commit de todas as alterações.
2. Envie (push) para o repositório principal no GitHub.
3. Vá em **Settings** > **Pages** no seu repositório.
4. Em **Build and deployment** > **Branch**, selecione `main` (ou sua branch principal) e a pasta `/ (root)`.
5. Clique em **Save**.
6. O jogo estará disponível em `https://<seu-usuario>.github.io/MonstroBolso/` em instantes.

// turbo
7. Verifique o status do deploy em **Actions**.
